Descrição do problema:

Verifique se um anagrama é válido para uma palavra fornecida

Anagramas são basicamente rearranjos na ordem de letras de palavras ou frases para formar outra palavra 
ou frase diferente 

Critérios:

* O anagrama não pode repetir letras

* O anagrama deve utilizar apenas as letras de word

* O anagrama deve ter o mesmo tamanho que word(palavra)

* o anagrama deve ser um anagrama, ou seja, não é aceita a mesma combinação inicial de letras de word, elas precisam
ser rearranjadas

Retorne true, se o anagrama passar por todos critérios definidos ou false caso falhe.

* Esta questão conta com 2 inputs (palavra,anagrama)

Exemplo 1:

Entrada: banana, nabana

Saída: true

Exemplo 2:

Entrada: rato,rato

Saida: false

Exemplo 3: 
Entrada: rato, toca
Saida: false
